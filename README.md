# 📈 Company Sales Analysis & Performance Tracking (Excel & Power Pivot)

A dynamic executive-level sales dataset performance dashboard engineered in Excel using a sleek dark theme mode to analyze monthly sales variances, team performance extremes, and MoM growth trajectories.

![Monthly Sales](Monthly%20Sales.png)

## 🎯 Project Objectives & Insights
* **Salesforce Evaluation:** Traced individual performance metrics per salesperson (e.g., John, Lisa, James) and isolated performance variance extremes.
* **Variance & Growth Analytics:** Built a Monthly Sales Variance waterfall chart alongside MoM Sales Analysis bars to pinpoint exact revenue fluctuations.
* **Multi-Dimensional Filtering:** Implemented deep slicing logic across Regions (North, South, West), Customer Segments (Retail, Wholesale), and Product Types.

## 🛠️ Tech Stack & Methods
* **ETL Engine:** Power Query (Transforming raw corporate transaction tables into clean dimensional structures).
* **Data Modeling:** Power Pivot (Handling tables relationships and metrics definition).
* **Formulas & Visuals:** DAX measures, Advanced Waterfall charts, custom ring charts for extreme performance tracking, and dynamic slicer panels.
